# Smart-Clothes-Protector-Arduino-Project
This project demonstrates an Arduino-based automatic clothes protection system using a rain water sensor and a servo motor. The main objective of the system is to protect clothes from getting wet by automatically moving them inside the house when rain is detected.

connection

Servo motor
Red +5v - 5v Arduino
Brown GND - GND Arduino
Yellow Signal - 9

Rain water Sensor to Sensor module

Sensor Module
VCC - 5V
GND - GND
DO - 2
